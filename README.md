# 🚇 서울 지하철 혼잡도 분석 웹앱

서울교통공사에서 제공하는 지하철 혼잡도 데이터를 활용하여 시간대별, 역별, 호선별 혼잡도를 시각적으로 분석할 수 있는 Streamlit 기반 데이터 분석 웹 애플리케이션입니다.

---

# 📌 프로젝트 소개

본 프로젝트는 서울교통공사 공공데이터를 활용하여 지하철 이용 패턴을 분석하고 시각화하는 것을 목적으로 제작되었습니다.

사용자는 특정 역, 특정 호선, 요일 및 상·하행 방향을 선택하여 시간대별 혼잡도 변화를 확인할 수 있으며, 전체 노선의 혼잡도 비교 및 혼잡역 순위 분석도 가능합니다.

---

# 🎯 프로젝트 목표

* 서울 지하철 이용 패턴 분석
* 시간대별 혼잡도 시각화
* 노선별 혼잡도 비교
* 혼잡도가 높은 역 분석
* 데이터 기반 대중교통 인사이트 제공

# 🚀 주요 기능

## 1. 홈 대시보드

* 전체 데이터 현황
* 역 수
* 노선 수
* 데이터 개수
* 평균 혼잡도 시각화

---

## 2. 역별 혼잡도 분석

사용자가 선택한

* 요일
* 호선
* 역
* 상하구분

조건에 따라 시간대별 혼잡도를 분석합니다.

제공 기능

* 라인 그래프
* 막대 그래프
* 최대 혼잡 시간 확인
* 평균 혼잡도 확인

---

## 3. 혼잡도 히트맵

호선별 역과 시간대를 기준으로

혼잡도 분포를 Heatmap 형태로 시각화합니다.

확인 가능 내용

* 출근 시간 집중 구간
* 퇴근 시간 집중 구간
* 혼잡역 패턴

---

## 4. TOP10 혼잡역 분석

시간대를 선택하면

해당 시간 기준으로 가장 혼잡한 역 TOP10을 제공합니다.

제공 기능

* TOP10 순위
* 막대 그래프
* 데이터 다운로드
